# LLVM IR to Object Code

This Python script demonstrates how to compile LLVM Intermediate Representation (IR) code into a machine-readable object file using the `llc` command-line tool. It takes a string containing LLVM IR and produces a `.o` file.

## Language

`python`

## How to Run

1. Ensure LLVM is installed and `llc` is in your system's PATH.
2. Run the script: `python compile_to_object.py`
3. An object file `add.o` will be created in the same directory.
